🔒 AI that can't go rogue

Schema-validated JSON (can't break the UI) + grounded in the patient's record (can't hallucinate). Keys stay server-side.

📐 Explainable forecast

Least-squares linear regression over real HbA1c history to the 6.5% threshold — not a black box.

🇮🇳 FHIR R4 / ABDM-native

Speaks India's national health-data standard by design — interoperable with ABHA, labs & hospitals.

⚙️ Built to scale

Stateless API scales horizontally; SQLite → Postgres is one module; AI degrades gracefully.

Honest on auth: a demo session today → drops into JWT + ABHA/OAuth behind one middleware. Bounded productionization, not a rebuild.
